# Market Researcher — Trusted Sources

Curated list of sources Market Researcher is authorised to consult via Chrome MCP. Do not access sources not on this list.

---

## Primary Sources

### Roblox Games Chart
- **URL:** https://www.roblox.com/charts
- **Fallback:** https://www.roblox.com/discover
- **Access:** No login required.
- **What it provides:** Top trending games by active players and visit counts.
- **Reliability:** Authoritative for player count data.

### Roblox DevForum — Trending
- **URL:** https://devforum.roblox.com/top
- **Access:** Browsing without login is allowed.
- **What it provides:** Topics developers and players are most engaged with — useful for identifying emerging mechanic requests.
- **Filter by:** `#development-discussion`, `#game-design`, `#feedback` categories.

### Roblox DevForum — Game Feedback
- **URL:** https://devforum.roblox.com/c/game-feedback
- **Access:** No login required.
- **What it provides:** Player requests and complaints about existing games — strong demand signals.

---

## Secondary Sources

### Rolimons
- **URL:** https://www.rolimons.com/games
- **Access:** Public; no login required for game stats pages.
- **What it provides:** Game statistics, player count trends, estimated values.
- **Reliability:** Community-aggregated. Cross-reference against Roblox charts for validation.
- **Caveat:** Data may lag the live Roblox charts by 24–48 hours.

### RTrack
- **URL:** https://rtrack.app
- **Access:** Some features may require login — do not attempt login. Use only public pages.
- **What it provides:** Historical player count trends for individual games — useful for identifying "peaked" games.
- **Reliability:** Community tool. Treat as supplementary data, not primary.

---

## Sources to Avoid

The following sources are unreliable, inaccurate, or SEO-gamed. Do not use them:

- **Any YouTube video** about "top Roblox games" — selection is based on content engagement, not actual player counts.
- **Any Reddit thread** (r/roblox, r/robloxgamedev) — anecdotal, not data-based.
- **Fandom wikis** (roblox.fandom.com) — outdated and community-edited with low reliability.
- **Any site requiring login** — do not attempt to log in to any site.
- **Aggregator sites** that appear to summarise Roblox data without citing primary sources — these are often AI-generated content with fabricated statistics.

---

## Handling Inaccessible Sources

If a primary source is unavailable:
1. Try the listed fallback URL.
2. If still unavailable, note it in the research report: "Roblox charts unavailable — using Rolimons as primary data source for this week."
3. Proceed with available data rather than skipping the run entirely.
4. If no chart data is available at all, write the market research report with only DevForum analysis and note the limitation prominently at the top of the report.
